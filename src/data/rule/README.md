# Canonical Programs from Rule et al., 2024

The file [`functions.csv`](functions.csv) is a copy of the canonical programs from Rule et al., 2024, downloaded from the OSF repository at [https://osf.io/gq2hj](https://osf.io/gq2hj), under the [`MIT license`](LICENSE).

The file [`functions.txt`](functions.txt) is an extracted version of the canonical programs from the CSV file, with proper formatting conversion, using the [`convert_rule_programs.py`](../../../scripts/convert_rule_programs.py) script.

**Citation for the Rule et al., 2024 paper:**

```
Rule, J.S., Piantadosi, S.T., Cropper, A., Ellis, K., Nye, M. and Tenenbaum, J.B., 2024. Symbolic metaprogram search improves learning efficiency and explains rule learning in humans. Nature Communications, 15(1), p.6847.
```
